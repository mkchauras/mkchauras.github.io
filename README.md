## Local Development

Post under work are inside the folder work-in-progress.

To serve on local end:

```bash
bundle config set path 'vendor/bundle'
bundle install && bundle exec jekyll serve
```

To add a new post

```bash
bundle exec jekyll post "<Post Title>"
```
